## Getting Started

### Prerequisites

To get started, make sure you have Node.js version 18 installed.

### Installation

1. Clone this repo
   ```
   git clone https://github.com/OumB2021/Computer-Store.git
   ```
2. Then, use NPM to install the projectâ€™s dependencies on ```root``` and ```front-end``` file:
   ```
   npm install
   ```
4. After that, run the development server
   ```
   npm run dev
   ```
5. Open http://localhost:3000 with your browser to see the result.
